Hide Deprecated Applications=Versteck veraltete Anwendungen
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Diese Einstellung versteckt veraltete Anwendungen in der Ansicht Verfügbaren Anwendungen aber werden unter Installierte und Vorherige Anwendungen noch angezeigt
Hide Incompatible Applications=Verstecke inkompatible Anwendungen
This setting will hide applications that are listed as being incompatible with your version of Unraid=Diese Einstellung versteckt Anwendungen die nicht mit dieser Unraid Version kompatibel sind
Enable Reinstall Default=Neuinstallation als Standard
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Diese Option aktiviert einen Button auf dem Pop-Up die eine bereits installierte Anwendung mit den Standardeinstellungen neuinstalliert. Diese Option sollte nur aktiviert werden wenn eine Anwendung mehrmals installiert werden soll
Enable additional search results from dockerHub=Weitere Suchergebnisse von DockerHub aktivieren
This setting enables CA to retrieve additional search results from dockerHub Note that docker must be enabled for this to be operational=Diese Einstellung erlaubt CA weitere Suchergebnisse von DockerHub zu empfangen Hinweis Der Docker Dienst muss hierfür aktiviert sein
Allow CA to send any emergency notifications=CA erlauben Notfall Benachrichtigungen zu senden
This setting allows CA to periodically scan your system and alert you to the presence of=
1 Known malware and security vulnerabilities within applications=1 Malware und Sicherheitslücke in Anwendung
2 Critical security vulnerabilities within Unraid OS=2 Kritische Sicherheitslücke in Unraid
3 Any installed applications which may seriously hinder the operation of your server=3 Installierte Anwendungen welche deinen Server vom ordnungsgemäßen Betrieb hindern
